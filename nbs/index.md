# Bienvenido a **Datos++**
## Un Puente entre la Teoría y la Práctica

**Datos++** es un libro digital interactivo diseñado para acompañarte en el aprendizaje de *Ingeniería de Datos*. Aquí encontrarás una integración fluida entre explicación teórica y práctica gracias al enfoque de *literate computing*, que combina texto, código ejecutable y resultados en un mismo entorno. El propósito es que aprendas de manera autónoma, paso a paso y a partir de problemas reales vinculados al trabajo profesional. Puedes acceder a la versión web del libro en [DatosPlusPlus](https://dci-courses.github.io/DatosPlusPlus/).

Cada unidad del libro reúne conceptos esenciales, ejemplos con código editable, ejercicios guiados y actividades de autoevaluación. Esto te permitirá experimentar, equivocarte, corregir y avanzar según tu propio ritmo, con un recurso pensado para apoyar tanto tus clases como tu estudio independiente. El contenido está estructurado de forma progresiva, de modo que cada sección amplía y profundiza la anterior.

**Datos++** también mantiene un repositorio público en GitHub, disponible en [repositorio GitHub](https://github.com/dci-courses/DatosPlusPlus). Allí podrás revisar el código fuente del libro, explorar ejemplos adicionales y proponer mejoras mediante el sistema de *issues*. El material puede descargarse para trabajar sin conexión. Esperamos que este libro se convierta en un punto de apoyo para desarrollar habilidades sólidas en análisis, transformación y modelado de datos. ¡Bienvenido!

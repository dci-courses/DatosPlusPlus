## Versión 1.0.0 (en progreso - 2025)

Primer lanzamiento oficial del libro **Datos++**.

### Contenido Incluido

- **Unidades fundamentales** de Ingeniería de Datos
- **Notebooks interactivos** con código ejecutable en Python
- **Ejercicios guiados** y actividades de autoevaluación
- **Material docente** para uso en clases

### Características Técnicas

- Integración con Jupyter Book para visualización web
- Soporte para ejecución local de notebooks
- Código exportable y reutilizable
- Compatibilidad con Python 3.10+

### Infraestructura

- Repositorio público en [GitHub](https://github.com/dci-courses/DatosPlusPlus)
- Despliegue automático con GitHub Pages
- Sistema de Dev Containers para desarrollo
